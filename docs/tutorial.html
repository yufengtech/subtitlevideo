使用教程 User Guide

轻松上手，充分利用即幕视频的强大功能 Get started and make the most of SubtitleVideo's powerful features

🚀 快速开始 🚀 Quick Start

  1. 导入您的视频文件 Import your video file
  2. 开始播放视频 Start playing the video
  3. 点击字幕按钮启用实时字幕 Tap the subtitle button to enable real-time subtitles
  4. 根据需要调整字幕设置 Adjust subtitle settings as needed

1 导入视频 Import Videos

支持的导入方式： Supported Import Methods:

  • 从相册选择：点击主界面的"选择视频"按钮，从相册中选择视频文件 From Photo Library: Tap "Select Video" button on main screen to choose from photo library
  • 文件分享：使用文件应用或其他应用分享视频到即幕视频 File Sharing: Share videos from Files app or other apps to SubtitleVideo
  • 拖拽导入：直接将视频文件拖拽到应用界面（iPad） Drag & Drop: Drag video files directly to app interface (iPad)

📁 支持的视频格式 📁 Supported Video Formats

2 视频播放 Video Playback

基本控制： Basic Controls:

  • 播放/暂停：点击视频中央或播放按钮 Play/Pause: Tap video center or play button
  • 快进/快退：拖拽进度条或使用手势 Seek: Drag progress bar or use gestures
  • 音量调节：使用设备音量键或系统控制中心 Volume: Use device volume keys or control center
  • 全屏播放：双击视频或点击全屏按钮 Fullscreen: Double-tap video or tap fullscreen button

💡 播放提示 💡 Playback Tips

为了获得最佳的字幕识别效果，建议确保视频音频清晰，背景噪音较少。 For best subtitle recognition, ensure video audio is clear with minimal background noise.

3 启用实时字幕 Enable Real-time Subtitles

第一次使用： First Time Setup:

  1. 播放视频后，点击屏幕显示控制界面 After playing video, tap screen to show controls
  2. 点击"字幕"按钮（CC图标） Tap "Subtitle" button (CC icon)
  3. 授权麦克风权限（用于音频处理） Grant microphone permission (for audio processing)
  4. 选择识别语言（中文/英文等） Select recognition language (Chinese/English etc.)

🎯 智能识别 🎯 Smart Recognition

应用会每5秒自动提取视频音频进行语音识别，生成的字幕会实时显示在视频底部。所有处理都在设备本地完成，保护您的隐私。 The app automatically extracts video audio every 5 seconds for speech recognition. Generated subtitles appear in real-time at the bottom of the video. All processing is done locally on your device to protect your privacy.

4 自定义设置 Customize Settings

字幕设置： Subtitle Settings:

  • 字幕大小：小、中、大、特大四种尺寸 Subtitle Size: Small, Medium, Large, Extra Large
  • 显示行数：设置同时显示的字幕行数（1-4行） Display Lines: Set number of subtitle lines (1-4 lines)
  • 识别语言：选择视频音频的主要语言 Recognition Language: Select primary language of video audio
  • 翻译语言：可选择将识别结果翻译为其他语言 Translation Language: Optional translation to other languages

应用设置： App Settings:

  • 界面语言：切换应用界面语言（中文/英文） Interface Language: Switch app interface language (Chinese/English)
  • 隐私设置：管理权限和数据使用偏好 Privacy Settings: Manage permissions and data usage preferences

5 高级功能 Advanced Features

音频提取： Audio Extraction:

应用内置强大的音频提取引擎，可以： Built-in powerful audio extraction engine that can:

  • 从任意时间点开始提取音频 Extract audio starting from any time point
  • 指定提取的时长 Specify extraction duration
  • 支持多种音频格式输出 Support multiple audio format outputs
  • 提供16kHz采样率的PCM格式音频 Provide 16kHz sample rate PCM format audio

批量处理： Batch Processing:

您可以连续处理多个视频文件，应用会记住您的设置偏好，提高工作效率。 You can process multiple video files continuously. The app remembers your settings preferences to improve efficiency.

6 故障排除 Troubleshooting

常见问题解决： Common Issue Solutions:

  • 字幕不显示：检查麦克风权限，确保视频有音轨 Subtitles not showing: Check microphone permission, ensure video has audio track
  • 识别不准确：选择正确的识别语言，确保音频清晰 Inaccurate recognition: Select correct recognition language, ensure clear audio
  • 应用卡顿：重启应用，释放设备内存空间 App freezing: Restart app, free up device memory
  • 视频无法播放：确认视频格式支持，检查文件完整性 Video won't play: Confirm video format support, check file integrity
